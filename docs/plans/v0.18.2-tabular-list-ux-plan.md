# v0.18.2 Tabular List UX Plan

## Summary

Convert list-heavy SecureTracker pages from card/block layouts to tabular views while keeping the Dashboard metric layout unchanged. Version `v0.18.2` is allocated under `AGENTS.md` because `origin/main` contains `v0.18.1`.

## Key Changes

- Applications, VAPT Calendar, Engagements, Organizations, and Users render data in responsive Material UI tables.
- Filters sit above tables and update rows based on search, status, type, organization, role, or year as relevant.
- Empty states appear inside table rows rather than leaving a blank page.
- Dashboard remains unchanged.

## Test Scenarios

- System Admin sees Applications, Calendar, Engagements, Organizations, and Users as tables.
- Filters/search update table rows on each relevant page.
- Create/update actions refresh tables without blank pages.
- Non-System roles do not see Users navigation.
- Dashboard still uses the existing metric-card layout.

## Assumptions

- Material UI table components are sufficient for MVP.
- Advanced grid features such as sorting, column pinning, and pagination are later enhancements.
