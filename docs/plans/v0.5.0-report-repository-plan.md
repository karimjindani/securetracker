# v0.5.0 Plan: Report Repository, MinIO Uploads, and Protected PDF Viewer

## Summary

Implement report storage and viewing for VAPT engagements. Reports are uploaded as PDFs, stored unchanged in MinIO, tracked with version metadata in PostgreSQL, audited on upload/view/download, and displayed from the engagement detail screen.

PDF passwords are never stored, logged, or submitted to the backend. Protected PDF passwords are entered only in the browser PDF viewer.

## Key Changes

- Add backend report repository APIs under engagements and reports.
- Store report files in MinIO object keys under `vapt-tracker/engagements/{engagementId}/reports/{reportId}/...`.
- Track report metadata through `Report` and `ReportVersion`.
- Store SHA-256 hash, file size, MIME type, version number, object key, upload notes, uploader, and protected flag.
- Detect password-protected PDFs conservatively by checking uploaded PDF bytes for encryption markers.
- Audit `REPORT_UPLOADED`, `REPORT_VERSION_UPLOADED`, `REPORT_VIEWED`, and `REPORT_DOWNLOADED`.
- Add engagement report panel, upload form, version list, authenticated view/download actions, and PDF.js viewer.
- Keep final reports immutable after engagement closure.
- Keep the Report Repository free of formal Scope Documents.

## Seeded and Regression Data

- No real PDFs are seeded by default.
- Regression tests generate temporary `REGRESSION_` PDF files at runtime.
- Cleanup removes report metadata and versions tied to `REGRESSION_` engagements and applications.
- Reset restores the v0.4.0 seeded baseline with no report files.

## Test Plan

- Run SDLC gates: `db:generate`, `typecheck`, `lint`, `test`, `build`, and `docker compose config --quiet`.
- Run regression gates: `test:e2e`, `test:regression`, `test:data:cleanup`, and `reset:seeded`.
- Verify draft upload advances `APPRISE_ASSESSMENT` to `DRAFT_REPORT_UPLOADED`.
- Verify protected PDF uploads set the protected flag without backend password handling.
- Verify unauthorized roles cannot upload reports.
- Verify download returns the original PDF bytes.
- Verify final report versions cannot be modified after engagement closure.
