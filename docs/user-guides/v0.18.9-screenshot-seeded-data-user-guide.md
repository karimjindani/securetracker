# v0.18.9 Screenshot Seeded Data User Guide

## Resetting Seeded Data

Run:

```powershell
npm.cmd run reset:seeded
```

The reset restores:

- 3 organizations: NBP, Paysys Labs, and Apprise
- 7 demo users
- 23 screenshot-derived applications
- 45 VAPT calendar engagements for 2026
- mixed Whitebox and Black/Grey assessment types

The seeded baseline does not create scoping records, reports, findings, risk acceptances, ticket records, or synthetic sample applications.

## Reviewing The Data

- Use Applications to review the 23 application names and owner/developer values from the tracker image.
- Use VAPT Calendar with Year `2026` and Starting Month filters to review month/type placement from the calendar image.
- Use Engagements Kanban to review the current status mapping derived from the pending-report tracker image.
