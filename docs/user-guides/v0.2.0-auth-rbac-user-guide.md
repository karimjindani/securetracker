# v0.2.0 Auth and RBAC User Guide

## Login

Start Keycloak, PostgreSQL, backend, and frontend using the deployment guide. Open:

```text
http://localhost:5173
```

Local Keycloak is available at:

```text
http://localhost:18080
```

Select **Login with Keycloak** and use a local demo account.

## Navigation

The portal displays navigation based on the authenticated user role. Backend API guards remain the source of truth for authorization.

## API Checks

Use a Keycloak access token to call:

```text
GET http://localhost:3000/me
GET http://localhost:3000/organizations
GET http://localhost:3000/users
```

Only `SYSTEM_ADMIN` can create or update users and organizations in this baseline.
