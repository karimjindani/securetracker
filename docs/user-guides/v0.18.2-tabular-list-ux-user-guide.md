# v0.18.2 Tabular List UX User Guide

## Tables

Applications, VAPT Calendar, Engagements, Organizations, and Users now show list data in tables. Use the filters above each table to narrow the visible rows.

## Page Behavior

- Applications can be searched by application, business owner, or technical owner.
- VAPT Calendar can be filtered by year and status.
- Engagements can be searched and filtered by status.
- Organizations can be filtered by name, type, and status.
- Users can be filtered by name/email, organization, role, and status.

## Dashboard

The Dashboard keeps its existing metrics and summary-card layout.
