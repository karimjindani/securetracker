# v0.1.0 User Guide

## Overview

`v0.1.0` is a platform foundation release. It includes the SecureTracker frontend shell, backend health endpoint, shared lifecycle constants, and local development services.

## Available UI

- Open `http://localhost:5173`.
- The shell displays the SecureTracker title, lifecycle summary, closure rule, report security summary, and audit principle.

## Available API

- Open `http://localhost:3000/health`.
- The endpoint returns service name, status, version, and first engagement status.

## Not Yet Available

- Login and RBAC screens.
- Application inventory.
- Engagement management forms.
- Report upload.
- Findings and revalidation workflows.
- Dashboards beyond the scaffold summary.
