# v0.3.0 User Guide: Applications and VAPT Calendar

## Applications

Use **Applications** to maintain the systems that can be included in VAPT planning.

Paysys Security Admin and System Admin users can add application records with:

- Application name
- Environment
- Criticality
- Business and technical owners
- URL
- Technology stack
- Internet-facing flag

Other portal roles with access can view application records for planning and governance visibility.

## VAPT Calendar

Use **VAPT Calendar** to create annual or ad-hoc planned VAPT entries.

NBP Security Admin, Paysys Security Admin, and System Admin users can create planned entries with:

- Application
- VAPT title
- Assessment type
- Planned month/year
- Optional planned start and end dates

Calendar entries created in this version remain in `PLANNED` status. Initiation, scoping, report upload, findings, closure, and Go-Live transitions are introduced in later versions.

## Access Rules

- Paysys Security Admin can manage application inventory.
- NBP Security Admin and Paysys Security Admin can manage planned calendar entries.
- Viewer, Developer, Vendor, and Auditor roles can view calendar/inventory records where permitted but cannot create them.
