# v0.3.1 User Guide: Ops Console

## Access

The Ops Console is available only in local/dev environments.

Required settings:

```powershell
$env:OPS_ENABLED="true"
$env:VITE_OPS_ENABLED="true"
```

Only `SYSTEM_ADMIN` can access `/ops`.

## Capabilities

The Ops Console shows backend, database, frontend, Keycloak, MinIO, SMTP test service, and Docker Compose status when available.

System Admin users can run the regression suite, clean regression-generated test data, and reset SecureTracker to seeded baseline data.

## Safety

Cleanup removes only records with the configured regression prefix. Reset removes business workflow records and restores seeded organizations. Keycloak users are still owned by the local realm import and local portal users are recreated on next login.
